Project Desription
The weather search application takes an zip code or city name as the input, verifies and retrieve information from Yahoo weather API and parsed into xml format data. It also returns to java servlet on AWS which transforms into JSON and finally displays as a user interface. Also, I integrated a functionality to post the weather information on Facebook using the Facebook SDK on web application.
Project Details
Input validation and retrieve information
Allow a user to enter a “query” to get weather information from Yahoo!; the query will contain a location which can be a city name with state or country, or a ZIP code. The app will have to determine the location type from the location input. If the user intends to input a city name, then state or country should be provided as well. Use the query string to retrieve appropriate information from Yahoo! The PHP script must run on AWS.

Display current and forecast weather details
Display the current weather details, along with a table showing five-day weather forecast. The big city name should be clickable and link to rss feed page. The underlined temperature unit should be clickable and perform the unit changing function.

Facebook Mashup
On the right side of the weather information, there should be a Facebook button, which allows the user to select to post either the current weather condition or the weather forecast to his/her Facebook wall; Authorize the user to login to Facebook. Post a feed of the weather information to the user’s Facebook wall using the Facebook Connect API.
