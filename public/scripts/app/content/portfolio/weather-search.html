Project Desription
This project involved creating a web based application that could display the current weather or a forecast for any entered zip code or city name and state/country code. I created a PHP application deployed on AWS that fetches the weather data from the yahoo service in form of an xml and then parses it it to extract the required information. It then creates another xml that is returned to a java servlet deployed on a Tomcat server which then transforms it to JSON form. These components together formed a REST web service that could now be invoked and consumed by any form of front end interface. I created the frontend as a web page in HTML and an Android application in java. For the webpage, i used AJAX to issue calls to the REST web service using javascript and formatted and displayed the result using YUI javascript library. For the android part, i used the Android SDK to call the REST web service and used native GUI components provided by Android to display the results. Lastly, i integrated a functionality to post the weather information on Facebook using the Facebook SDK on both web and Android interfaces.
